认证接口

POST /api/auth/login 用户登录获取 Token

请求参数

参数名类型必填说明
usernamestring必填用户名
passwordstring必填密码

请求示例

curl -X POST {{baseUrl}}/api/auth/login \ -H "Content-Type: application/json" \ -d '{"username":"your_username","password":"your_password"}'

响应示例

{ "success": true, "token": "eyJhbGciOiJIUzI1NiIs...", "user": { "id": 1, "username": "your_username" } }
POST /api/auth/register 用户注册

请求参数

参数名类型必填说明
usernamestring必填用户名（唯一）
passwordstring必填密码
emailstring可选邮箱地址

记忆管理接口

GET /api/memories 获取记忆列表

请求头

Authorization: Bearer YOUR_API_KEY

查询参数

参数名类型必填说明
pagenumber可选页码，默认1
limitnumber可选每页数量，默认20
searchstring可选搜索关键词
tagsstring可选标签过滤（逗号分隔）

请求示例

curl -X GET "{{baseUrl}}/api/memories?limit=10" \ -H "Authorization: Bearer YOUR_API_KEY"
POST /api/memories 创建新记忆

请求参数

参数名类型必填说明
contentstring必填记忆内容
titlestring可选记忆标题
tagsarray可选标签数组
importancenumber可选重要性 1-5
categorystring可选分类

请求示例

curl -X POST "{{baseUrl}}/api/memories" \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "content": "今天学习了 Node.js 的异步编程模式", "title": "Node.js 学习笔记", "tags": ["编程", "Node.js", "学习"], "importance": 4 }'
GET /api/memories/:id 获取单个记忆详情

请求示例

curl -X GET "{{baseUrl}}/api/memories/123" \ -H "Authorization: Bearer YOUR_API_KEY"
PUT /api/memories/:id 更新记忆

请求示例

curl -X PUT "{{baseUrl}}/api/memories/123" \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{"content": "更新后的内容", "importance": 5}'
DELETE /api/memories/:id 删除记忆

请求示例

curl -X DELETE "{{baseUrl}}/api/memories/123" \ -H "Authorization: Bearer YOUR_API_KEY"

智能搜索接口

POST /api/search 语义搜索记忆

请求参数

参数名类型必填说明
querystring必填搜索查询
limitnumber可选返回数量限制
thresholdnumber可选相似度阈值 0-1

请求示例

curl -X POST "{{baseUrl}}/api/search" \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{"query": "关于编程的学习笔记", "limit": 5}'

IDE 集成教程

与 Trae IDE 集成

在 Trae IDE 中配置 777-MS 记忆系统：

// 在 ~/.trae-cn/settings.json 中添加 { "memory": { "enabled": true, "apiUrl": "{{baseUrl}}", "apiKey": "YOUR_API_KEY" } }

配置完成后，Trae 会自动将对话中的重要信息存储到记忆系统。

通过 MCP 协议集成：

// 在 ~/.trae-cn/mcp-config.json 中添加 { "mcpServers": { "777-ms-memory": { "command": "node", "args": ["path/to/777-ms/mcp-server.js"], "env": { "API_URL": "{{baseUrl}}", "API_KEY": "YOUR_API_KEY" } } } }

与 Cursor IDE 集成

// 在 Cursor 设置中添加自定义 API { "cursor.ai.customApis": { "memory": { "url": "{{baseUrl}}/api", "headers": { "Authorization": "Bearer YOUR_API_KEY" }, "endpoints": { "search": "/search", "create": "/memories", "list": "/memories" } } } }

与 VS Code 集成

// 安装扩展后，在 settings.json 中配置 { "memoryExtension.apiUrl": "{{baseUrl}}", "memoryExtension.apiKey": "YOUR_API_KEY", "memoryExtension.autoSave": true, "memoryExtension.contextAware": true }

通用 HTTP 调用示例

// JavaScript/Node.js 示例 const API_URL = '{{baseUrl}}'; const API_KEY = 'YOUR_API_KEY'; // 创建记忆 async function createMemory(content) { const response = await fetch(`${API_URL}/api/memories`, { method: 'POST', headers: { 'Authorization': `Bearer ${API_KEY}`, 'Content-Type': 'application/json' }, body: JSON.stringify({ content }) }); return response.json(); } // 搜索记忆 async function searchMemories(query) { const response = await fetch(`${API_URL}/api/search`, { method: 'POST', headers: { 'Authorization': `Bearer ${API_KEY}`, 'Content-Type': 'application/json' }, body: JSON.stringify({ query }) }); return response.json(); }
# Python 示例 import requests API_URL = '{{baseUrl}}' API_KEY = 'YOUR_API_KEY' HEADERS = { 'Authorization': f'Bearer {API_KEY}', 'Content-Type': 'application/json' } # 创建记忆 def create_memory(content): response = requests.post( f'{API_URL}/api/memories', headers=HEADERS, json={'content': content} ) return response.json() # 搜索记忆 def search_memories(query): response = requests.post( f'{API_URL}/api/search', headers=HEADERS, json={'query': query} ) return response.json()
